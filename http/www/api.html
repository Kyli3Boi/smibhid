SMIBHID - API

Endpoints

  • MAC address (GET): /api/wlan/mac
  • Firmware version (GET): /api/version
  • List sensor modules (GET): /api/sensors/modules
  • List module sensors (GET): /api/sensors/{module}
  • Get latest module sensor readings (GET): /api/sensors/readings/{module}
  • Get latest sensor readings from all modules (GET): /api/sensors/readings
  • Get last 60 minutes (minute) or hourly roll up log (hour) of sensor readings from all modules (GET): /api/sensors/readings/log/{log_type} - minute or hour
  • Firmware files (GET,POST): /api/firmware_files - List, add or remove URLs staged for download and patch on reset
  • Get SCD30 measurement status (GET): /api/sensors/scd30/auto_measure
  • Put SCD30 measurement status (PUT): /api/sensors/scd30/auto_measure/{start_stop}
    • List files: GET
    • Add file: POST: Submit Value = "Add", URL = Str: id = "url"
    • Remove file: POST: Submit Value = "Remove", Str: URL
  • Reset (POST): /api/reset - reset SMIBHID
