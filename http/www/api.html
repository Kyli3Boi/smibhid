SMIBHID - API

Endpoints

  • MAC address (GET): /api/wlan/mac
  • Firmware version (GET): /api/version
  • Download (POST): /api/download - Download a file from a URl to the update staging directory
  • Update (POST): /api/update - Copy files in the update staging directory to the lib folder
  • Reset (POST): /api/reset - reset SMIBHID
