SMIBHID - API

Endpoints

  • MAC address (GET): /api/wlan/mac
  • Firmware version (GET): /api/version
  • Firmware files (GET,POST): /api/firmware_files - List, add or remove URLs staged for download and patch on reset
    • List files: GET
    • Add file: POST: Submit Value = "Add", URL = Str: id = "url"
    • Remove file: POST: Submit Value = "Remove", Str: URL
  • Reset (POST): /api/reset - reset SMIBHID
